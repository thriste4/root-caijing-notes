# root-caijing-notes

本仓库用于归档和发布多个独立的 HTML 页面。这些页面内容涉及财经相关领域的笔记、资料整理或信息展示，不针对任何具体网站或域名。

## 项目简介

这是一个以文档归档为主要目的的仓库。所有内容均以静态 HTML 文件形式存放，可直接通过 GitHub Pages 或其他静态托管服务访问。仓库本身不涉及数据库、后端服务或动态交互功能。

## 目录说明

```
├── index.html          # 仓库首页 / 导航页（如有）
├── pages/              # 存放主要 HTML 页面文件
│   ├── page-1.html
│   ├── page-2.html
│   └── ...
├── assets/             # 静态资源（CSS、JS、图片等）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md           # 本文件
```

- `pages/`：存放所有独立 HTML 页面，按需命名。
- `assets/`：存放页面引用的公共静态资源。
- 根目录下的 `index.html` 可作为导航或入口页面（如存在）。

## 页面归档说明

- 每个 HTML 页面为独立完整文件，可单独打开使用。
- 页面之间不依赖特定服务器环境，纯前端静态。
- 内容均为公开可查阅的资料整理，不包含任何形式的营销、诱导或商业承诺。
- 页面内不包含任何具体网站、域名或关键词推广。
